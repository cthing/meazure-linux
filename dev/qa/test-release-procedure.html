Meazure Test and Release Procedure

Prerequisites

  1. VM running Ubuntu 20.04
  2. Two screens
  3. X11 window system
  4. GNOME desktop
  5. Create a release candidate package 
    ./build.sh -t release clean
./build.sh -t release project_full_build
  6. Publish a candidate release on the GitHub project
    • Release: major.minor.patch RCn
    • Tag: major.minor.patch-RCn
    • Title: major.minor.patch RCn
    • Description: "See [CHANGELOG.md](CHANGELOG.md)"
    • Upload Debian package and SHA256 file
    • Check box next to "Set as a pre-release"

Clean Installation and Startup

  1. Verify that Meazure is not installed by running 
    dpkg-query -l meazure
    If Meazure is not installed, the following is displayed: 
    dpkg-query: no packages found matching meazure
    If Meazure is installed, the following is displayed: 
    Desired=Unknown/Install/Remove/Purge/Hold
| Status=Not/Inst/Conf-files/Unpacked/halF-conf/Half-inst/trig-aWait/Trig-pend
|/ Err?=(none)/Reinst-required (Status,Err: uppercase=bad)
||/ Name           Version      Architecture Description
+++-==============-============-============-================================================================
ii  meazure        5.0.0-1      amd64        A tool for easily measuring and capturing portions of the screen
  2. If Meazure is installed, remove it by running 
    sudo dpkg -r meazure
  3. Delete the directory ~/.config/CThingSoftware
  4. Download the candidate release Debian package from the GitHub project
  5. Install the Debian package by running: 
    sudo dpkg -i package
  6. After installation is complete, verify the following files exist: 
    ls -l /usr/bin/meazure /usr/share/meazure/help/help.html \
      /usr/share/meazure/icu/icudt72l.dat \
      /usr/share/man/man1/meazure.1.gz \
      /usr/share/applications/org.cthing.meazure.desktop \
      /usr/share/metainfo/org.cthing.meazure.metainfo.xml \
      /usr/share/mime/packages/org.cthing.meazure.xml \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure.svg \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure-config.svg \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure-positions.svg \
      /usr/share/doc/meazure/copyright
  7. Verify the Meazure icon appears as a desktop application
  8. Add the Meazure icon as a Favorite
  9. Click on the Meazure icon in the Favorites bar
  10. Verfiy that Meazure program user interface is displayed
  11. Verify that the Help > Help menu item is enabled
  12. Exit Meazure
  13. Verify that the configuration file exists: 
    ls -l ~/.config/CThingSoftware/Meazure.conf
  14. Launch Meazure from a console: 
    meazure
  15. Verify the program is displayed and there are no error or warning messages in the console
  16. Select File > Export Configuration... item
  17. Specify a directory and name for the configuration file
  18. Once the file is saved, exit Meazure
  19. Open the desktop's graphical file browser
  20. In the file browser, go to the directory containing the exported configuration file
  21. Verify the file is displayed with the Meazure configuration file icon
  22. Double-click the icon
  23. Verify that Meazure is launched
  24. Record a position by pressing Ctrl+P
  25. Select File > Save Positions As...
  26. Specify a directory and name for the position log file
  27. Once the file is saved, exit Meazure
  28. In the file browser, go to the directory containing the saved position log file
  29. Verify the file is displayed with the Meazure position log file icon
  30. Double-click the icon
  31. Verify that Meazure is launched
  32. Exit Meazure
  33. In a console, run: 
    sudo dpkg -r meazure
  34. Verify that the Meazure icon has been removed from the Favorites bar
  35. Verify that the following files have been removed: 
    ls -l /usr/bin/meazure /usr/share/meazure/help/help.html \
      /usr/share/meazure/icu/icudt72l.dat \
      /usr/share/man/man1/meazure.1.gz \
      /usr/share/applications/org.cthing.meazure.desktop \
      /usr/share/metainfo/org.cthing.meazure.metainfo.xml \
      /usr/share/mime/packages/org.cthing.meazure.xml \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure.svg \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure-config.svg \
      /usr/share/icons/hicolor/scalable/apps/org.cthing.meazure-positions.svg \
      /usr/share/doc/meazure/copyright
  36. Reinstall the Debian package by running: 
    sudo dpkg -i package

Cursor Tool

  1. Select the Cursor Tool using the toolbar
  2. Verify the Status bar says "Cursor hotspot is measurement point"
  3. Verify the Tools > Cursor menu item indicates selected
  4. Verify the X1 and Y1 text fields are enabled but not editable
  5. Verify the X1 and Y1 up/down buttons are enabled but do not do anything
  6. Verify a number cannot be pasted into the X1 or Y1 fields
  7. Verify all other fields in the Tool Information section are disabled
  8. Verify the Data window appears next to the cursor
  9. Verify the Data window shows the X1 and Y1 coordinates of the cursor

  11. Verify that the Data window position changes to avoid being hidden
  12. Verify the Magnifier tracks the cursor position and the red square is over the cursor hotspot
